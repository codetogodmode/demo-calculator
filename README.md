# demo-calculator

Martinova referenční implementace Calculatoru pro Code to God Mode akademii.

Tento repo obsahuje kód který Martin píše **live na sessions**. Postupně v průběhu Layer 1 a Layer 2 se Calculator rozvíjí — od prvního Hello Worldu až po refaktorovanou OOP verzi.

## Jak to funguje

Na každé session Martin otevře issue z kanbanu (Projects → Calculator), implementuje odpovídající věci v tomto kódu, commitne a zavře issue.

Memberové:
- Sledují tento repo pro referenční Calculator implementaci
- Mají vlastní Calculator v jejich `member-{name}/src/Calculator/` — stejný kód, jen píšou ho sami
- Rozvíjí PERSONAL projekt offline (PandaNursery, PlaylistManager, ...) podle úkolů v jejich kanbanu

## Jak spustit

```bash
dotnet run --project src/Calculator
```
